\begin{frame}{まとめ}
  \begin{itemize}
    \item $\tan x = k$ は $0 \leq x < \pi$（または $0 \leq x < 2\pi$）の範囲で解を求める
    \item 三角不等式はグラフを描いて $y \geq k$ または $y \leq k$ の範囲を読む
    \item 定義域の制限（$\tan x$ が未定義の点）を忘れずに除く
  \end{itemize}
